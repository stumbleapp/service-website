# Stumble Service Website

https://stumble.app/  

Powered by [Vite](https://vitejs.dev/) and [React](https://react.dev/).  
NOTE: Nodejs >= 22 is necessary.  

## Building the Site

The site is built automatically with every push thanks to [Cloudflare Pages](https://pages.cloudflare.com/).  
If you want to build it locally, follow this simple step:  

```
npm install && npm run dev
```

You can now edit the files in the `src` directory and the site will reflect the changes immediately thanks to live reloading.
